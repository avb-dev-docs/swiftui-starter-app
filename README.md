#  SwiftUI Starter App

This repository contains a number of SwiftUI views to accelerate iOS and macOS development. 

* Multiplatform SwiftUI Starter App, deploys to iOS and macOS devices
* Multiple databases
  * Core Data
  * RealmDB
* Components
  * Table
